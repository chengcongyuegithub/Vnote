# 并查集完成二叉树的祖先问题
首先给了标准的二叉树结构,然后就是Query类,我们来看一下这个Query类
```
public class Query
{
  public Node o1;
  public Node o2;
  public Query(Node o1,Node o2)
  {
      this.o1=o1;
      this.o2=o2;
  }
}
```

首先我们生成两个map,queryMap和indexMap
queryMap表示的当前key为o1,对应的就是o2,如果是o2的话,对应的就是o1.
