# 运算符
## 运算符优先级
```
public void complicatedexpression_r(){
    int x=20, y=30;
    boolean b;
    b = x > 50 && y > 60 || x > 50 && y < -60 || x < -50 && y > 60 || x < -50 && y < -60;
    System.out.println(b);
}
```
根据优先级,题中符号的优先级排序是：'>'，'<'，'&&'，'||',先大于,后小于....先与在或
```
 b = (x > 50 && y > 60) || (x > 50 && y < -60) || (x < -50 && y > 60) || (x < -50 && y < -60);
```
