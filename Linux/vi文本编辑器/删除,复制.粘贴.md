# 删除,复制.粘贴
## 删除
先贴自己比较熟悉的
```
在当前模式下
x,向后删除一个字符
X,向前删除一个字符

dd删除这一行
ndd删除向下的多少行
```
按照书中进行补充
```
dG 当前到最后一行
d1G 当前到第一行
```
```
d0 当前行的游标到最开始
d$ 当前行的游标到最后
```
## 复制和粘贴
粘贴就只有一个操作就是P和p,和X和x对应,P是当前位置向前粘贴,p是当前位置向后复制.
最常用的就是p,然后我们来介绍一下复制的操作,复制和删除相对应
```
yy复制当前这一行
nyy复制当前这一行到最后

yG
y1G

y0
y$
```
## 还原和重做
两个比较重要的操作
```
u是还原
ctrl+r是重做
.是重复做上面的步骤
```
